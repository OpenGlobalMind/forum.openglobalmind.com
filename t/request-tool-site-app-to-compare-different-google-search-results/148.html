Request: Tool/site/app to compare different Google Search results

Discussions and Actions Action! Doing Things Now
skreutzer August 7, 2020, 11:33pm #1

The request is to build a tool/site/app/plugin which, for a search phrase, displays the Google Search results side by side based on different search configurations, namely (but not limited to) the search for exact matches (in quotation marks) and the similar, broader matches if the search is done without quotation marks around the phrase. The purpose is to become able to show any regular user the differences in the results, especially for learning and teaching tech/tool/media competency, so in schools for example students could gain understanding about how to effectively search for material on the Web and what is to be expected from different approaches.

Could be a very quick, simple prototype hacked together, other than that a browser plugin was suggested, main application would be in a school class.
